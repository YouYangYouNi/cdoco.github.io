# register_streaming_read_callback( string|array|function $callback )

Register a callback function to execute whenever a data stream is read from using <CFRequest::streaming_read_callback()>.

Register a callback function to execute whenever a data stream is read from using <CFRequest::streaming_read_callback()>.

The user-defined callback function should accept three arguments:

  • <span class="php-var">$curl_handle</span> - resource - Required - The cURL handle resource that represents the in-progress transfer.
  • <span class="php-var">$file_handle</span> - resource - Required - The file handle resource that represents the file on the local file system.
  • <span class="php-var">$length</span> - integer - Required - The length in kilobytes of the data chunk that was transferred.

Parameters

$callback
string|array|function
$callback (Required) The callback function is called by <php:call_user_func()>, so you can pass the following values:
  • The name of a global function to execute, passed as a string.
  • A method to execute, passed as <span class="php-keyword1">array</span>(<span class="php-quote">'ClassName'</span>, <span class="php-quote">'MethodName'</span>).
  • An anonymous function (PHP 5.3+).
