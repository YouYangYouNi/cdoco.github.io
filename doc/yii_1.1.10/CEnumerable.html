CEnumerable是所有枚举类的基类。

定义一个枚举类型，继承CEnumberable并定义字符串常量。 每一个枚举值代表一个常数。 常量的名字必须与常量的值相同。 例如， 
class TextAlign extends CEnumerable
{
    const Left='Left';
    const Right='Right';
}
然后， 可以使用TextAlign::Left和TextAlign::Right的枚举值。
